# Minishell
Implementation of a "mini" shell.

Features:
* Argument parsing
* Several builtin-commands including: echo, exit, cd, shift, unshift, stat, read
* Command substitution
* Environment variable processing
* Variable Expansion
* Main argument processing
* Wildcard expansion
* Signal Processing
* Redirection
* Piping
* Statements

## Usage
### Compilation 
```bash
make
```

### Clean
```bash
make clean
```
### Execution
```bash
./msh
```
